这个博客使用以下技术构建：

- **SCSS** - CSS 预处理器

- **GitHub Pages** - 免费托管

## 为什么选择 Astro？

Astro 是一个现代化的静态站点生成器，它有以下优点：

1. **极快的性能** - 默认零 JavaScript
2. **内容优先** - 专为博客和文档设计
3. **灵活** - 支持多种 UI 框架
4. **简单** - 学习曲线平缓
